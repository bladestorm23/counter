Firebase setup

Step 1 — Create Firebase project
  1. Go to console.firebase.google.com → Add project → name it "taps"
  2. Build → Authentication → Get started → enable Email/Password
  3. Build → Firestore Database → Create database → production mode
Step 2 — Firestore security rules

In Firestore → Rules, paste and publish:

rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /taps/{userId}/{doc=**} {
      allow read, write: if request.auth.uid == userId;
    }
  }
}

Step 3 — Paste your config
  1. Gear → Project settings → Your apps → Add app → Web
  2. Copy the three values from your firebaseConfig
